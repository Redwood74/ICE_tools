This document provides a plain-language summary of the legal structure
governing ICEpicks. It is not legal advice. For specific legal questions,
consult a licensed attorney.

## License summary

ICEpicks is licensed under the **ICE Advocacy Public License (IAPL) v1.0**.
See [`../LICENSE.md`](../LICENSE.md) for the full text.

**Key characteristics:**

- **Source-available, not open source.** The source code is publicly
  readable and forkable, but this is not an OSI-approved open-source license.
  Standard open-source freedoms do not apply.

- **Non-commercial use is free.** Personal use, legal aid, academic
  research, civil-rights oversight, and related humanitarian uses are
  permitted without payment or permission.

- **Categorical prohibition on enforcement use.** No person or entity may
  use ICEpicks to assist immigration enforcement, detention, surveillance,
  or related activities. This prohibition is absolute.

- **Forks allowed under conditions.** Derivative works must use the same
  license, use distinct branding, and be non-commercial.

- **Commercial use requires a paid license.** Contact
  `licensing.icepicks@rqn.com`.

## Absolute prohibitions

No license (including commercial licenses) permits:

- Use by ICE, CBP, DHS, or any immigration enforcement agency for
  enforcement operations
- Use to surveil, track, or locate individuals for enforcement purposes
- Use to facilitate detention, deportation, or removal
- Any use that materially assists immigration enforcement against individuals

These prohibitions cannot be waived or overridden by any license.

## Verification process

Organizations seeking a formal License Reference ID should submit a
verification request to `licensing.icepicks@rqn.com`. See
[`../FAQ.md`](../FAQ.md) for details.

The verification process:
1. Applicant submits organization info, website, intended use
2. Licensor reviews (typically within 10 business days)
3. Applicant receives a one-time verification code (8-char base32, 7-day expiry)
4. Applicant confirms via verification link
5. Licensor issues a License Reference ID
